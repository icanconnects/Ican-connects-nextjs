# ICAN Next.js Website (Next 15 + Tailwind)

This project is a modern, minimal-professional community website for the Indian Community Association of North York (ICAN).
It uses Next.js 15 (App Router) + Tailwind CSS and is ready for deployment on Vercel.

## Quick start (locally)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Deploy to Vercel
1. Push this repository to GitHub.
2. Import repository in Vercel (vercel.com) and deploy. Vercel auto-detects Next.js projects.

Replace placeholder contact info and events with real content before publishing.
